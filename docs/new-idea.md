## 🕌 Core UX Concept

**Philosophy:** Not a chatbot — this is a *knowledge lens*. The metaphor is a **scholarly reading room**, not a chat window. Users orient themselves spatially through Islamic themes, then dive deep via semantic search.

**User flow on first landing:**
1. The hero presents a Qur'anic ayah immediately (establishes spiritual tone)
2. The search bar is prominent but framed as an *inquiry field*, not a chat prompt
3. Suggested queries guide general users; thematic tiles guide da'i and students
4. Results surface as verse cards — readable, referenceable, expandable

## 🔍 Example Search Interaction

**User types:** *"How to invite non-Muslims to Islam gently"*

The platform returns:
- **An-Nahl 16:125** — the da'wah methodology verse (hikmah, maw'iza)
- **Fussilat 41:33** — excellence of those who call to Allah
- Each card shows Arabic text → translation → surah reference → theme label → "Explore tafsir →"

## 🎨 Visual Style

| Element | Choice |
|---|---|
| **Typography** | Lateef (Arabic), Cormorant Garamond (Latin) — elegant, scholarly |
| **Palette** | Warm vellum background · Deep teal · Burnished gold accents |
| **Atmosphere** | Like an illuminated manuscript meets a modern research archive |
| **Motion** | Gentle staggered fade-ins; no aggressive animations |
| **Tone** | Calm, trustworthy, reverent — suitable for both casual visitors and scholars |
